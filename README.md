# コウテイちゃん — 構成と公開手順

毎日の小さな問いかけに答えると、コウテイちゃん（あたたかい母性キャラ）が
やさしくフィードバック。自己肯定感をすこしずつ育てるアプリ。

## ファイル構成
```
index.html      アプリ本体（UI + ロジック + コウテイちゃん）
manifest.json   PWA設定
sw.js           オフライン動作
icon-192/512    アイコン（コウテイちゃんの顔）
```

## 設計のキモ（第二段階への布石）

`index.html` の中に、2つの「隔離された層」があります。
ここを差し替えるだけで、UIを触らずに進化できます。

1. **store**（データ層）— いまは localStorage。将来 Supabase 等へ。
2. **kotei**（フィードバック層）— いまはルールベースの返事。
   第二段階で AI(API) に差し替えるなら、この `kotei.daily()` /
   `kotei.weekly()` の中身だけ置き換えればOK。UIは無変更。

## 質問構成（第一段階）
- 日次（スライド式・6問）：①自分との約束 ②人格的によかった行動
  ③ネガティブ感情を否定しない ④小さな成功 ⑤人への貢献 ⑥ひとり時間
- 週次（2問）：⑦人生の柱8本の充実度 ⑧柱がなくても良かったこと10個
- 回答後、コウテイちゃんが日次・週次それぞれにフィードバック

## 公開（更新の反映）
GitHub に上げたファイルを差し替えれば、GitHub Pages のURLに自動反映。
1. 変更した index.html などを GitHub にアップロード（上書き）
2. 1〜2分で公開URLに反映される
