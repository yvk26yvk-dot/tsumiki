# UX改善スプリント 実装計画

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 日次チェックの日付指定、タブ「ふりかえり」→「こんしゅう」改名、きろくの月別アコーディオン化、きろくのカレンダービュー(design b)を実装する。

**Architecture:** 単一の純関数 `buildCalendar(year,month,opts)` を①日次の日付ピッカーと④きろくのカレンダーで共用。きろくのカード描画を `dailyCardHtml()` に共通化し、リスト表示とカレンダーの日タップ詳細で再利用。store層・kotei層・認証・同期は一切変更しない。

**Tech Stack:** Vanilla JS(index.html単一ファイル・ビルドなし)。純関数 `buildCalendar` のみ node で単体テスト可能。

**Spec:** `docs/superpowers/specs/2026-07-16-ux-sprint-design.md`

## Global Constraints

- コウテイちゃんの人格: あたたかい母性・タメ口・責めない
- 配色は既存CSS変数(--rose #E0907F / --rose-deep #C26B58 / --rose-soft / --leaf #8FA079 / --leaf-deep / --ink 系)のみ
- カレンダーは**日曜はじまり**(既存WD配列 `['日','月','火','水','木','金','土']` と整合)
- 日付キーは既存 `key(d)`→'YYYY-MM-DD'、`fmtDate(d)`、`weekKey(d)`→'W'+週の月曜 を流用
- ①遡り無制限・未来日は選択不可・デフォルト今日。保存済み過去日は完了画面→書き直す
- ④はdesign b: 記録日は日付マスを薄ローズの丸で塗り、選択日は--rose-deepソリッド丸。件数ドットなし。日タップ詳細は**既存きろくカードを流用**
- アコーディオン開閉・ビュー選択・深夜プロンプト選択は**セッション内のみ**(localStorage永続化しない)
- store層・kotei層・認証・設定・同期ロジックは変更禁止
- sw.js の CACHE はリリース時に `kotei-v8`(現在v7)
- コミットは各タスク末尾。**pushしない**
- ブランチ: `feature/ux-sprint` を main から作成

**実装順序(ユーザー指定):** ②改名 → buildCalendar基盤 → ①日付指定 → ③アコーディオン → ④カレンダー → 検証。④は最後。

```js
function ymd(y,m,d){return y+'-'+String(m).padStart(2,'0')+'-'+String(d).padStart(2,'0');}
/* 月グリッドのHTML(日曜はじまり)。opts:{selKey,maxKey,mark}。クリック可能セルはdata-cal-day属性を持つ */
function buildCalendar(year,month,opts){
  opts=opts||{};
  const selKey=opts.selKey||null, maxKey=opts.maxKey||null, mark=opts.mark||null;
  const first=new Date(year,month-1,1).getDay();      /* 0=日 */
  const daysIn=new Date(year,month,0).getDate();
  const head='<div class="cal-wd">'+WD.map((w,i)=>'<span class="'+(i===0?'sun':i===6?'sat':'')+'">'+w+'</span>').join('')+'</div>';
  let cells='';
  for(let i=0;i<first;i++) cells+='<div class="cal-cell"></div>';
  for(let d=1;d<=daysIn;d++){
    const k=ymd(year,month,d);
    const future=maxKey && k>maxKey;
    const sel=selKey && k===selKey;
    const has=mark && mark(k);
    let cls='cal-cell';
    if(future)cls+=' future';
    let inner;
    if(sel) inner='<span class="cal-day sel">'+d+'</span>';
    else if(has) inner='<span class="cal-day has">'+d+'</span>';
    else inner='<span class="cal-day'+(future?' fut':'')+'">'+d+'</span>';
    cells+='<div class="'+cls+'"'+(future?'':' data-cal-day="'+k+'"')+'>'+inner+'</div>';
  }
  return head+'<div class="cal-grid">'+cells+'</div>';
}
```

```js
function dailyCardHtml(dateKey,e,filter){
```
